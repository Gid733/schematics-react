Schematics generators for React

## Installation
- npm:
  ```
  npm install -g @angular-devkit/schematics-cli
  npm install --save-dev schematics-react
  ```

- yarn:
  ```
  yarn global add @angular-devkit/schematics-cli
  yarn add -D schematics-react
  ```
